# Moving to Denver Project

## Summary

- This is a project to map useful geographic data in Denver, CO.
- The major points of interest are:
  - Hospitals/Clinics
  - New Housing, by price ~500k
  - Apartments / Rentals

## Setup

`pip install folium`
`pip install -U homeharvest`
